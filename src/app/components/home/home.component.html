BIZmat alpha

BIZmat is the world's first library to provide business specific component level UI solutions to make developers' life easy and managers' track record un-scathed.

We would love for you to go through our documentation and try it out in your own application.
Any feedback will be appreciated.



Get Started

Install

BIZmat uses angular-material as base to build its components

npm install bizmat --save
ng add @angular/material

Import

Once BIZmat has been installed, make sure you import the BizmatModule and add it to imports as shown below

import { BizmatModule } from 'bizmat';
...
NgModule({
 imports: [
  ...
  BizmatModule
 ],
 ...
})

Using Components:

<bizmat-COMPONENT_NAME (COMPONENT_EVENT_EMITTER) = "CALLBACK_FUNCTION()" [COMPONENT_PROPERTY] = "VALUE_VARIABLE" ></bizmat-COMPONENT_NAME>

You can find further information on each of the components and their usage in their dedicated pages.
